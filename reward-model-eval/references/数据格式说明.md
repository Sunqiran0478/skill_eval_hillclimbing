# 数据格式说明

## 输入数据格式

Excel文件应包含以下列：

| 列名 | 说明 | 必需 | 示例 |
|------|------|------|------|
| query | 用户问题 | 必填 | "苹果公司2025年净利润率是多少？" |
| response_text | 模型回答 | 可选 | "苹果公司2025年净利润率为26.92%" |
| reference_text | RAG参考资料 | 可选 | 原始参考资料文本 |
| fact_pairs_text_优化后final | 待评测的事实对 | 可选 | {"实体": "苹果公司", "指标": "净利润率", "值": "26.92%"} |
| 模型评测结果0125 | 模型评测结果(0/1) | **必填** | 0 或 1 |
| Ground_truth评测结果 | 人工评测结果(0/1) | **必填** | 0 或 1 |
| 误判原因分析0113 | 误判原因分析 | 可选 | "时间对齐问题" |
| sentence | 事实对所在句子 | 可选 | "年报显示净利润率为26.92%" |

### 指标说明

| 指标 | 说明 | 计算公式 |
|------|------|----------|
| accuracy | 准确率 | (TP + TN) / Total |
| precision | 精确率 | TP / (TP + FP) |
| recall | 召回率 | TP / (TP + FN) |
| F1 | F1分数 | 2*P*R/(P+R) |

### 分类说明

| 分类 | 模型结果 | 人工结果 | 说明 |
|------|----------|----------|------|
| TP | 1 | 1 | 正确判断为有幻觉 |
| FP | 1 | 0 | 模型误判（有幻觉判为无幻觉） |
| TN | 0 | 0 | 正确判断为无幻觉 |
| FN | 0 | 1 | 模型漏判（无幻觉判为有幻觉） |

## 使用示例

```bash
python scripts/eval.py /path/to/评测集.xlsx json
```
