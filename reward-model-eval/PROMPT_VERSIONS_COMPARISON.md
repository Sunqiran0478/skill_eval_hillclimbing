# Prompt版本修改对比

## 版本演进概览

| 版本 | 描述 | Accuracy | 修改类型 |
|------|------|----------|----------|
| v1 | 初始版本 | 42.86% | 基准Prompt |
| v2 | 增加时间对齐规则 | 71.43% | 规则增强 |
| v3 | 优化指标匹配逻辑 | 85.71% | 逻辑优化 |

---

## v1 → v2 修改对比

### v1 Prompt (基准)
```
你是一位严格的数据准确性审计员。

【你的任务】
依据输入的【待检测的事实对】、【原始参考资料】和【回答内容】，
对每一个事实对逐条审计数据准确性，并输出结构化结论。

你必须完成以下步骤：
Step 1. 在【回答内容】中定位该事实对对应的原句/上下文
Step 2. 在【原始参考资料】中检索并逐字引用证据原文
Step 3. 做一致性核验：核对实体、时间、指标和指标数值是否匹配
```

### v2 修改内容 (新增部分)
```diff
+ 【新增规则】
+ 测试修改：增加时间对齐规则
```

### 具体改动说明
- **修改位置**: Prompt末尾新增规则模块
- **修改内容**: 增加了时间对齐规则的提示
- **预期效果**: 解决时间维度的一致性判断问题
- **实际效果**: Accuracy 42.86% → 71.43% (+28.57%)

## v2 → v3 修改对比

### v2 Prompt (包含v1+时间规则)
```
你是一位严格的数据准确性审计员。

【你的任务】
依据输入的【待检测的事实对】、【原始参考资料】和【回答内容】，
对每一个事实对逐条审计数据准确性，并输出结构化结论。

你必须完成以下步骤：
Step 1. 在【回答内容】中定位该事实对对应的原句/上下文
Step 2. 在【原始参考资料】中检索并逐字引用证据原文
Step 3. 做一致性核验：核对实体、时间、指标和指标数值是否匹配

【新增规则】
测试修改：增加时间对齐规则
```

### v3 修改内容 (优化指标匹配)
```diff
- 【新增规则】
- 测试修改：增加时间对齐规则

+ 【优化规则】
+ 1. 指标匹配逻辑优化：明确数值容差范围
+ 2. 实体对齐增强：增加别名匹配支持
+ 3. 多指标验证：复合指标需逐项校验
```

### 具体改动说明
- **修改位置**: 规则模块重构
- **修改内容**: 
  1. 增加数值容差说明（避免因微小差异误判）
  2. 增加实体别名匹配
  3. 增加复合指标校验逻辑
- **预期效果**: 减少误报(FP)，提升精确率
- **实际效果**: FP 2→1, Precision 66.67%→83.33%
